Calculator App
```
A simple and user-friendly calculator built using HTML, CSS, and JavaScript. The project was created to practice layout and DOM manipulation.

Features:
- Basic arithmetic operations: + − × ÷
- Keyboard input support
- AC (clear all) and CE (delete last character) buttons
- +/- operation
- Percentage %
- Protection from double operators (e.g., ++, --, //)
- Error handling with Invalid Entry message
- Technologies used: HTML5, CSS3, JavaScript (Vanilla JS)

How to run
Simply open the index.html file in any browser.

Project structure
calculator/
  index.html
  style.css
  script.js
  fonts/
    digital-7.ttf
```
